Self-Parking Robot

This was my capstone project for MTE-121 (Digital Computation). We wrote some RobotC code and used a few different sensors to program an EV3 Mindstorm robot to actively search for a suitable parking spot in it's line of sight and perform a parallel park.


OrganizeEm'

This was my first web application I made after taking CS50X. OrganizeEm' is a minimalistic web application that lets you enter and view tasks, and has cool features like letting you organize tasks based on priority, and uses SQL to store the actual tasks.
